## IEEE Project Report

This folder contains the **IEEE-style academic report** associated with the *Ontario Highway Traffic Patterns and Congestion Trends* data visualization project.

### Report Overview
The report documents:
- Problem definition and objectives
- Dataset description and preprocessing
- Visualization methodology and design choices
- Key findings and insights derived from the analysis

The report follows **IEEE formatting and reporting standards**, complementing the interactive Power BI dashboard by providing methodological detail and academic context.
